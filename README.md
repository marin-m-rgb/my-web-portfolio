# Personal Portfolio
A clean, modern landing page designed to showcase design services and skills.

## About the Project
This project was developed to demonstrate my ability to build clean, responsive front-end interfaces. The focus was on creating a modern UI/UX experience that is both visually engaging and highly functional.

## Design & Features
* **Modern Aesthetics:** Designed with a refined color palette (using modern CSS variables) to ensure a professional and cohesive brand identity.
* **Typography:** Leveraged Google Fonts (Paytone One & Montserrat) to create a clear visual hierarchy.
* **Responsive Layout:** Built with a focus on usability, ensuring the interface looks great across different screen sizes.

## Tech Stack
* **HTML5** & **CSS3** (Modern layout techniques like Flexbox).
* **Design System:** Standardized variables for colors and fonts for easier maintainability.
* **Workflow:** Version control managed via GitHub.

## Key Learnings
During this project, I focused on:
* Implementing clean code practices through CSS variables.
* Optimizing user interface elements to improve readability and visual impact.
* Applying UI/UX principles to ensure a seamless navigation experience.
